Structure de sélection multiple

Syntaxe algorithmique


            Selon  sélecteur Faire
                valeur 1 : < Séquence d’instructions 1>
                valeur 2 : < Séquence d’instructions 2>
                … 
                valeur n : < Séquence d’instructions n>
            Sinon 
                <Autre Séquence d’instructions>
            FinSelon

La séquence d’instructions numéro i sera exécutée si la valeur du sélecteur appartient à la ième liste de valeurs. Le sélecteur est une variable ou une expression de type scalaire (un entier ou un caractère).

Exemple


            Selon age Faire
                2: ecrire("Salut bebe !");
                6: ecrire("Salut gamin !");
                12: ecrire("Salut jeune !");
                16: ecrire("Salut ado !");
                18: ecrire("Salut adulte !");
                68: ecrire("Salut papy !");
                else
                    ecrire("Je n'ai aucune phrase de prete pour ton age ");
            FinSelon

Syntaxe En C


            switch(variable) //au cas où la variable vaut:
            {
                case valeur1: 
                    //si variable=valeur1: executer ce bloc d'instructions…
                    break;
                case valeur2:
                    //si variable= valeur2: executer ce bloc d'instructions…
                    break;
                //……
                default: 
                    //si aucune des valeurs precedentes: executer ce bloc d'instructions, pas de "break" ici.
            }

Le bloc "default" n'est pas obligatoire.

Exemple


            switch (age)
            {
                case 2:
                    printf("Salut bebe !");
                    break;
                case 6:
                    printf("Salut gamin !");
                    break;
                case 12:
                    printf("Salut jeune !");
                    break;
                case 16:
                    printf("Salut ado !");
                    break;
                case 18:
                    printf("Salut adulte !");
                    break;
                case 68:
                    printf("Salut papy !");
                    break;
                default:
                    printf("Je n'ai aucune phrase de prete pour ton age ");
                    break;
            }

Remarque

L’exemple précédent peut être résolu en utilisant plusieurs instructions « Si » imbriquées, mais l’algorithme sera très lourd :


            Si( age= 2) alors
                ecrire("Salut bebe !")
            Sinon Si ( age= 6) alors
                ecrire("Salut gamin !")
            Sinon Si ( age= 12) alors
                ecrire("Salut jeune !")
            Sinon Si ( age= 16) alors
                ecrire("Salut ado !")
            Sinon Si ( age= 18) alors
                ecrire("Salut adulte !")
            Sinon Si ( age= 68) alors
                ecrire("Salut papy !")
            else
                ecrire("Je n'ai aucune phrase de prete pour ton age ")
            FinSi
