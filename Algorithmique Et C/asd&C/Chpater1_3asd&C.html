Premier Exemple

Problème

Écrire un programme qui permet de calculer et d’afficher la surface d’un rectangle.

Nb : La surface = la longueur * la largeur

Solution

  1. Étape 1 : Analyse
  1. Étape 2 : Mon premier algorithme
        
Algorithme Surface
Variables
  Lo : Entier
  La : Entier
Debut
  Ecrire(« Donner la longueur »)
  Lire(Lo)
  Ecrire(« Donner la largeur »)
  Lire(La)
  S ← Lo * La
  Ecrire(« La surface est », S)
Fin
  1. Etape 3 : Mon premier programme (traduction-C)
        
#include 
int main() {
    int lo, la, S;

    // Demander la longueur
    printf("Donner la longueur : ");
    scanf("%d", &lo);

    // Demander la largeur
    printf("Donner la largeur : ");
    scanf("%d", &la);

    // Calculer la surface
    S = lo * la;

    // Afficher la surface
    printf("La surface est : %d\n", S);

    return 0;
}
