Tableaux multidimensionnels

Les tableaux à deux dimensions se représentent comme une matrice ayant un certain nombre de lignes (première dimension) et un certain nombre de colonnes (seconde dimension). Nous pouvons représenter schématiquement un tableau de 3 lignes et de 4 colonnes comme suit

Déclaration en algorithmique


Types
    typeMatrice : Tableau[1..nbLignes,1..nbColonnes] de Type
Variables
    nomMatrice : typeMatrice

Exemple


Types
NMat : Tableau[1..3,1..4] de Réel
Variables
Matrice : NMat

Déclaration en Programmation C


type nom[dim1][dim2]; /* dim1 lignes et dim2 colonnes */

Exemple

int Matrice [3][4] ; /* tableau de 3 lignes et 4 colonnes */

Utilisation

Un élément du tableau à deux dimensions est repéré par ses indices. Pour accéder à chaque case du tableau il faut écrire : nom[indice1][indice2]

Tout comme un élément d’un tableau à 1 dimension, tableau[i][j] fait référence à l’élément de la ligne i et de la colonne j de tableau et se manipule comme n’importe quelle variable. Remarque

  • En algorithmique les tableaux à deux dimensions commencent aux indices 1. Les indices maximum sont donc dim1-1, dim2-1.
    • En langage C les tableaux à deux dimensions commencent aux indices 0. Les indices maximum sont donc dim1-1, dim2-1.

Exemple

En algorithmique


Matrice[2][1]<--5;
Ecrire( Matrice [2][1])
lire(Matrice [1][2])

Matrice [2][1]désigne l’élément situé à la 2ème ligne et la 1ère colonne.

En langage C


int Matrice [2][3] ;
Matrice [1][0] = 5;
printf("%d", Matrice[1][0]);
scanf("%d",& Matrice [0][1]);

Matrice[1][0]désigne l’élément situé à la 2ème ligne et la 1ère colonne.

Application 1 : Remplissage et affichage d’un tableau à deux dimensions

Pour remplir les lignes et les colonnes des tableaux à deux dimensions, nous utiliserons deux indices (p.ex: I et J), et la structure for, souvent imbriquée,

En algorithmique

Types

Variables

   dim1 ,dim2: Entier

	i ,j: Entier
Début
    Pour i de 1 à lignes

            // Demander à l'utilisateur d'entrer une valeur

        Fin Pour
    Fin Pour

    Fin Pour

En Programmation C


#include 
void RemplirTableauMultidimensionnel (float matrice[][], int lignes , int colonnes) {
int i, j;
    for (i = 0; i < lignes; i++) {
        for (j = 0; j < colonnes; j++) {
            // Demander à l'utilisateur d'entrer une valeur
            printf("Entrez la valeur pour tableau[%d][%d] : ", i+1, j+1);
            // Lire la valeur entrée par l'utilisateur et la stocker dans le tableau
            scanf("%f", &matrice[i][j]);
        }
    }
}

void AfficherTableauMultidimensionnel(float matrice[][], int lignes, int colonnes) {    int i, j;
    printf("Les éléments du tableau sont :\n");
    for (i = 0; i < lignes; i++) {
        for (j = 0; j < colonnes; j++) {
            printf("La valeur pour tableau[%d][%d] = %.2f\n", i+1, j+1, matrice[i][j]);
        }
    }
}

int main() {

    printf("Entrez le nombre de colonnes du tableau : ");
    scanf("%d", &dim2);

Application 2 : Recherche séquentielle d’un élément dans un tableau multidimensionnel

Voici une procédure une procédure de recherche séquentielle qui vise à localiser un élément x dans un tableau T. 

Procédure RechercherElementDansMatrice(int matrice[][], int lignes, int colonnes), x: Entier)
Variable
    i,j: Entier
    trouve: Booléen
Début
    ébut
    // Initialiser les indices et la variable trouvé
    i ← 1
    j ← 1
    trouve ← Faux

    // Tant qu'on n'a pas trouvé l'élément et qu'on n'a pas parcouru toutes les lignes
    Tant que (i <= lignes) ET (trouve = Faux)
        // Parcourir les colonnes de la ligne i
        Tant que (j <= colonnes) ET (trouve = Faux)
            Si matrice[i][j] = x Alors
                trouve ← Vrai
           Sinon
                j ← j + 1
            Fin Si
        Fin Tant que
        // Revenir à la première colonne après avoir parcouru la ligne entière
        j ← 1
        i ← i + 1
    Fin Tant que

    Si trouve = Vrai Alors
         Ecrire("Élément trouvé à la position [", i, "][", j, "]") 
    Sinon
    // Si l'élément n'a pas été trouvé
        Ecrire("Élément non trouvé dans le tableau.")
    Fin Si
Fin RechercherElementDansMatrice
