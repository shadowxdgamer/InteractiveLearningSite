Les tableaux à une dimension

Les tableaux sont une suite de variables de même type situées dans un espace contigu en mémoire ;

Lorsqu'un tableau est créé, il prend un espace contigu en mémoire : les cases sont les unes à la suite des autres.

Toutes les cases d'un tableau sont du même type. Ainsi, un tableau de int contiendra uniquement des int, et pas autre type.

 Un tableau a une dimension bien précise. La fig. suivante est un schéma d'un tableau de 4 cases en mémoire. Ces 4 cases doivent être contiguës.

Déclaration en algorithmique


Variables
Nom_tab : Tableau [PremInd..DernInd] de Type_éléments

Exemple


Tnote : Tableau[1..30] de Réel

Remarque

Il est également possible de définir un type tableau comme dans l’exemple suivant :


Constantes
    n = 10
Types
    Tab = Tableau[1..n] de Entier
Variables
    T : Tab

Remarque

En algorithmique, le premier élément d'un tableau porte l'indice 1

Déclaration en Programmation C

type nom[dim];

Exemples


int compteur[4];  /* le compilateur réserve des places pour 4 entiers.*/
float Tnote [30];   /* le compilateur réserve des places pour 30 réels. */

Remarque

En programmation C, le premier élément d'un tableau porte l'indice 0

Utilisation

Un élément du tableau est repéré par son indice. Pour accéder à chaque case du tableau il faut écrire :

nom[indiceDeLacase]

Exemples


//L'instruction suivante affecte à la variable X la valeur du premier élément du tableau Note :
       X ← Note[1]
//L'instruction suivante affiche le contenu du quatrième élément du tableau Note :
       Ecrire(Note[4])
//L'instruction suivante affecte une valeur introduite par l'utilisateur à l'élément trois du tableau Note :
      Lire(Note[3])

Application 1 : Remplissage et affichage d’un tableau

Le fait que les éléments d'un tableau soient indicés permet de remplir ou afficher tous ces éléments avec une boucle, en utilisant une variable qui sert d'indice et s'incrémente à chaque tour de boucle.

En algorithmique


Algorithme RemplissageEtAffichageTableau
Types
   Tab = Tableau[1..100] de Entier
Variables
   T : Tab
   taille: Entier


Procédure RemplirTableau(VarT: Tab, taille: Entier)
Variable
	i: Entier
Début
        Pour i de de 1 à taille faire
            Ecrire ( "Entrez la valeur pour l'élément ", i, " : ")
            Lire (T[i])
        Fin Pour
 Fin RemplirTableau



Procédure AfficherTableau(T: Tab, taille: Entier)
Variable
    i: Entier
Début
    Ecrire("Les éléments du tableau sont : ")
    Pour i de de 1 à taille faire
        Ecrire("T[", i , "] = ", T[i])
    Fin Pour
Fin AfficherTableau


Début
    Ecrire("Entrez la taille du tableau : ")
    Lire(taille)    
    // Appel de la procédure pour remplir le tableau
   RemplirTableau(T, taille)
    // Appel de la procédure pour afficher le tableau
   AfficherTableau(T, taille)
Fin

En Programmation C


#include
void RemplirTableau(int T[], int taille) {
    int i;  // Déclaration de la variable i
    // Boucle pour parcourir chaque élément du tableau
    for (i = 0; i < taille; i++) {
        printf("Entrez la valeur pour l'élément %d : ", i + 1); // Afficher l'invite
        scanf("%d", &T[i]);  // Lire la valeur et la stocker dans le tableau
    }
}


void AfficherTableau(int T[], int taille) {
    int i;  // Déclaration de la variable i    
    // Affichage du message initial
    printf("Les éléments du tableau sont : \n");
    
    // Boucle pour parcourir chaque élément du tableau
    for (i = 0; i < taille; i++) {
        // Afficher chaque élément du tableau avec son numéro
        printf("T[%d] = %d\n", i + 1, T[i]);
    }
}


int main() {
    int taille;
    int T[100];  
    // Demande de la taille du tableau
    printf("Entrez la taille du tableau : ");
    scanf("%d", &taille);
// Déclaration du tableau de taille variable
    // Appel de la procédure pour remplir le tableau
    RemplirTableau(T, taille);
    return 0;
}

Application 2 : Recherche séquentielle d’un élément dans un tableau

L'algorithme suivant effectue une recherche séquentielle dans un tableau. Il commence à l'index 1, et compare chaque élément du tableau à l'élément x. Si l'élément est trouvé, l'algorithme affiche la position de l'élément et arrête la recherche. Sinon, il continue jusqu'à ce que tous les éléments du tableau soient parcourus. Si l'élément n'est pas trouvé, un message d'échec est affiché. 


Procédure RechercherElement(T: Tab, taille: Entier, x: Entier)
Variable
    i: Entier
    trouve: Booléen
Début
    trouve ← Faux  // Initialisation de la variable trouve à Faux
    Tant que (i<=n Et trouve=Faux) Faire
        Si T[i] = x Alors
            trouve ← Vrai
	Sinon
	ii+1
        Fin Si
Fin Tant que
    Si trouvé = Vrai Alors
            Ecrire("L'élément ", x, " a été trouvé à la position ", i)
Sinon
        Ecrire("L'élément ", x, " n'a pas été trouvé dans le tableau.")
    Fin Si
Fin RechercherElement

En Programmation C


void RechercherElement(int T[], int taille, int x) {
    int i = 0;  // Initialisation de la variable i à 0 (pour parcourir le tableau)
    int trouve = 0;  // Variable booléenne (0 = Faux, 1 = Vrai)

    // Boucle Tant que i est inférieur à taille et que l'élément n'est pas trouvé
    while (i < taille && trouve == 0) {
        if (T[i] == x) {  // Si l'élément à la position i est égal à x
            trouve = 1;  // L'élément a été trouvé
        } else {
            i++;  // Passer à l'élément suivant
        }
    }

    // Vérification si l'élément a été trouvé
    if (trouve == 1) {
        printf("L'élément %d a été trouvé à la position %d\n", x, i + 1);  // Affichage avec la position
    } else {
        printf("L'élément %d n'a pas été trouvé dans le tableau.\n", x);  // Message si l'élément n'a pas été trouvé
    }
}
