Modes de passage de paramètres

La substitution des paramètres effectifs aux paramètres formels s’appelle passage de paramètres. Elle correspond à un transfert d’informations entre le programme ou le module appelant et la procédure appelée.

Il y a deux méthodes de passage de paramètres dans une procédure : le passage par valeur et le passage par variable. Ces méthodes sont décrites ci-dessous.

Passage par valeur

Dans ce type de passage, le paramètre formel reçoit uniquement une copie de la valeur du paramètre effectif. La valeur du paramètre effectif ne sera jamais modifiée.

Exemple

En algorithmique 

Algorithme Passage_par_valeur
Variables 
    N : entier
    //Déclaration de la procédure P1
    Procédure P1(A : entier)
    Début
        A ← A * 2
        Ecrire(A)
FinProc
//Algorithme principal
Début
   N ← 5
   P1(N)
   Ecrire(N)
Fin

Cet algorithme définit une procédure P1 pour laquelle on utilise le passage de paramètres par valeur.

Lors de l'appel de la procédure, la valeur du paramètre effectif N est recopiée dans le paramètres formel A. La procédure effectue alors le traitement et affiche la valeur de la variable A, dans ce cas 10.

Après l'appel de la procédure, l'algorithme affiche la valeur de la variable N dans ce cas 5.

La procédure ne modifie pas le paramètre qui est passé par valeur.

En programmation C

Voici la traduction de l'algorithme "Passage_par_valeur" en langage C :


#include 
// Déclaration de la procédure P1
void P1(int A) {
    A = A * 2;
    printf("%d\n", A);  // Affiche la valeur de A
}

// programme principal
int main() {
    int N = 5;
    P1(N);             // Appel de la procédure P1 avec N comme paramètre
    printf("%d\n", N); // Affiche la valeur de N après l'appel de la procédure
    return 0;
}

Passage par référence ou par adresse

Dans ce type de passage, la procédure utilise l'adresse du paramètre effectif. Lorsqu'on utilise l'adresse du paramètre, on accède directement à son contenu. La valeur de la variable effectif sera donc modifiée.

En algorithme les paramètres passés par adresse sont précédés du mot clé Var.

Exemple1

En algorithmique

Reprenons l'exemple précédent :


Algorithme Passage_par_référence
Variables 
N : entier
//Déclaration de la procédure P1
Procédure P1 (Var A : entier)
Début
    A ← A * 2
    Ecrire(A)
FinProc
//Algorithme Principal
Début
      N ← 5
      P1(N)
      Ecrire(N)
Fin

A l'exécution de la procédure, l'instruction Ecrirer(A) permet d'afficher à l'écran 10. Au retour dans l'algorithme principal, l'instruction Ecrire(N) affiche également 10.

Dans cet algorithme le paramètre passé correspond à la référence (adresse) de la variable N. Elle est donc modifiée par l'instruction :

A <-- A *2

En programmation C

En C, le passage par référence se fait en passant l'adresse de la variable (à l'aide du symbole &) dans la fonction.

Dans la fonction P1, le paramètre est un pointeur (int *A), ce qui permet de modifier directement la valeur de N.

L'opérateur * est utilisé pour accéder à la valeur de la variable à laquelle pointe le pointeur.

#include 
// Déclaration de la procédure P1 (passage par référence en C)
void P1(int *A) {
    *A = *A * 2;         // Modifie la valeur de A via son adresse
    printf("%d\n", *A);   // Affiche la nouvelle valeur de A
}
// Algorithme principal
int main() {
    int N = 5;
    P1(&N);              // Appel de la procédure P1 avec l'adresse de N
    printf("%d\n", N);   // Affiche la valeur de N après l'appel de P1
    return 0;
}

Exemple2

En algorithmique

Voici un exemple de procédure en C qui calcule la puissance d'un entier a élevé à une puissance b .


Procédure puissance(a, b : Entier ; Var c : Entier)
Variables
    i : Entier 
Début
    c <--1
    Pour i de 1 à b Faire
        c <-- c * a
    FinPour
Fin.

En Programmation C


    #include 
// Déclaration de la procédure puissance
void puissance(int a, int b, int *c) {
    *c = 1; // Initialisation de c à 1
    for (int i = 1; i <= b; i++) {
        *c = (*c) * a; // Multiplie c par a, b fois
    }	
}

// Algorithme principal pour tester la procédure
int main() {
    int a, b, c;
    
    printf("Entrez la base (a) : ");
    scanf("%d", &a);
    
    printf("Entrez l'exposant (b) : ");
    scanf("%d", &b);
    
    puissance(a, b, &c); // Appel de la procédure avec c passé par référence
    printf("La puissance de %d élevé à %d est : %d\n", a, b, c); // Affiche le résultat
    
    return 0;
}

Explcation

La fonction puissance prend trois paramètres : a, b (les entiers de la base et de l'exposant) et un pointeur vers c, où le résultat sera stocké.

La valeur de c est passée par référence en utilisant un pointeur (int *c), ce qui permet à la procédure de modifier directement la variable c dans la fonction appelante.

La boucle for multiplie c par a, b fois, réalisant ainsi le calcul c à la puissance a.

Exemple d'exécution :

Si vous entrez a = 2 et b = 3, le programme affichera :

La puissance de 2 élevé à 3 est : 8

Exemple 3

La procédure « permut » permet d’échanger les valeurs de 2 entiers a et b.

En algorithmique

Procédure permut(Var a, b : Entier)
Variables
    x : Entier 
Début
    x <-- a 
    a <-- b 
    b <-- x
Fin

En Programmation C

#include 
// Déclaration de la procédure permut (passage par référence)
void permut(int *a, int *b) {
    int x;
    x = *a;  // Stocke la valeur de a dans une variable temporaire x
    *a = *b; // Affecte la valeur de b à a
    *b = x;  // Affecte la valeur de x (ancien a) à b
}
// Algorithme principal pour tester la procédure
int main() {
    int a, b;    
    // Lecture des valeurs de a et b
    printf("Entrez la valeur de a : ");
    scanf("%d", &a);    
    printf("Entrez la valeur de b : ");
    scanf("%d", &b);    
    // Appel de la procédure permut
    permut(&a, &b); // Passe les adresses de a et b    
    // Affichage des valeurs après permutation
    printf("Après permutation, a = %d, b = %d\n", a, b);    
    return 0;
}

Explication

La procédure permut prend deux pointeurs int *a et int *b comme paramètres, ce qui permet de modifier directement les variables a et b dans le programme principal.

L'échange des valeurs se fait à l'aide d'une variable temporaire x. La valeur de a est stockée dans x, puis la valeur de b est affectée à a, et enfin la valeur de x (l'ancienne valeur de a) est affectée à b.

Dans le programme principal, les adresses de a et b sont passées à la procédure avec l'opérateur &.

Exemple d'exécution

Si vous entrez a = 3 et b = 7, le programme affichera :

Après permutation, a = 7, b = 3
