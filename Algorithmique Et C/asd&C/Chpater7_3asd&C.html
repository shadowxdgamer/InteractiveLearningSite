Opérations sur les chaînes de caractères

La concaténation

C’est l’assemblage de deux chaînes de caractères en utilisant l’opérateur « + »en algorithmique et la fonction strcat()de string.h en C

Exemple

En algorithmique


chn1 <-- ”Programmation”
chn2 <-- ”Java”
chn3 <-- chn1+” ”+chn2

La variable chn3 contiendra ”Programmation Java

En programmation C


#include 
#include   // Nécessaire pour manipuler les chaînes de caractères

int main() {
    char chn1[10] = "Turbo";    // Déclare une chaîne de caractères de taille 10 pour "Turbo"
    char chn2[10] = "Pascal";     // Déclare une chaîne de caractères de taille 10 pour "Pascal"
    char chn3[20];                // Déclare une chaîne de caractères de taille 20 pour contenir la combinaison

    // Copie chn1 dans chn3
    strcpy(chn3, chn1);
    // Ajoute un espace à la suite de chn3
    strcat(chn3, " ");
    // Ajoute chn2 à la suite de chn3
    strcat(chn3, chn2);

    // Affiche le résultat
    printf("chn3: %s\n", chn3);
    return 0;
}

les opérateurs relationnels (>, >=, <, <=, =, #)

En algorithmique, Il est possible d’effectuer une comparaison entre deux chaînes de caractères, le résultat est de type booléen. La comparaison se fait caractère par caractère de la gauche vers la droite selon le code ASCII.

Exemples

  • L’expression (”a” > ”A”) est vraie puisque le code ASCII de ”a” (97) est supérieur à celui de ”A” (65)
  • L’expression (”programme” < ”programmation”) est fausse puisque ”e” > ”a”
  • L’expression (”” = ” ”) est fausse (le vide est différent du caractère espace).

en C la comparaison entre deux chaînes de caractères se fait à l'aide de la fonction strcmp()de la string.h.

Exemples


#include 
#include

    if ('a' > 'A') {
        printf("L'expression ('a' > 'A') est vraie.\n");
    } else {
        printf("L'expression ('a' > 'A') est fausse.\n");
    }
    
    if (strcmp("programme", "programmation") < 0) {
        printf("L'expression ('programme' < 'programmation') est vraie.\n");
    } else {
        printf("L'expression ('programme' < 'programmation') est fausse.\n");
    }
 
    if (strcmp("", " ") == 0) {

Accès à un caractère dans une chaîne

Pour accéder à un caractère de la chaîne, il suffit d’indiquer le nom de la chaîne suivi d’un entier entre crochets qui indique la position du caractère dans la chaîne.

Exemple


chn<-- ”Programmation Java”
c<--chn[7]

la variable c contiendra le caractère ”m”.

En général, ch[i] désigne le ième caractère de la chaîne ch.
