La boucle Répéter … Jusqu’à

Syntaxe algorithmique :


Répéter
     <Séquence d’instructions>
Jusqu’à <condition>

Principe de fonctionnement

La séquence d’instructions est exécutée une première fois, puis l’exécution se répète jusqu’à ce que la condition de sortie soit vérifiée. Une boucle « répéter » s’exécute toujours au moins une fois. En effet, le test se fait à la fin.

Exemple1


i<--1
Répéter
    ecrire("Salut mes chers etudiants! ");
    i<--i + 1
Jusqu’à (i > 10)

Cette boucle répète 10 fois l'affichage de «Salut mes chers etudiants ! »

Exemple2


i<--1
Répéter
    Ecrire(i*10)
     i<--i + 1
Jusqu’à (i > 5)

Cette boucle affiche respectivement les nombres 10, 20, 30, 40 et 50.

Exemple3


i<--5
Répéter
    Ecrire(i*10)
     i<--i - 1
Jusqu’à (i < 1)

Cette boucle affiche respectivement les nombres 50, 40, 30, 20 et 10.

Exemple4


i<--0
Répéter
    Ecrire ("La variable compteur vaut ", compteur);
    compteur<-- compteur + 1
Jusqu’à(compteur>=10)

Remarque

Contrairement à une boucle « pour », dans une boucle « répéter », l’initialisation et l’avancement du compteur doivent être gérés manuellement par le programmeur. Dans une boucle « répéter », il faut toujours s’assurer que la condition de sortie sera vérifiée après un nombre fini de parcours. Sinon, c’est une boucle infinie comme dans le cas suivant : 


c<--”A”
Répéter
    Ecrire(c)
Jusqu’à (c > ”Z”)

Syntaxe en C

Il n’y a pas une correspondance exacte en C , bien que ce type de boucle est très similaire à Repeter.. tant Que et non pas à Repeter.. Jusqu’à Il s'agit de l'instruction: 


Do
{
    /* bloc d’instructions*/
}While(expression);

Exemple1


i = 1;
do
{
    printf("Salut mes chers etudiants !\n");
    i++;
} 
while (i < =10);

Exemple2


i = 1;
do
{
	printf("%d ",i*10) ;
    i++;
} 
while (i < =5);

    i--;
