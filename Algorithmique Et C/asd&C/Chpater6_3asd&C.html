Recherche dichotomique

La recherche dichotomique, est un algorithme efficace utilisé pour rechercher un élément dans un tableau trié. Contrairement à une recherche séquentielle, qui vérifie chaque élément un par un, la recherche dichotomique divise le tableau en deux parties à chaque étape, ce qui permet de réduire considérablement le nombre de comparaisons nécessaires.

Principe de fonctionnement

Soient premier et dernier les extrémités gauche et droite de l’intervalle dans lequel on cherche la valeur x, on calcule M, l’indice de l’élément médian : M = (premier + dernier) div 2 Il y a 3 cas possibles :

  • x = T[M] : l’élément de valeur x est trouvé, la recherche est terminée
  • x < T[M] : l’élément x, s’il existe, se trouve dans l’intervalle [premier..M-1]
  • x > T[M] : l’élément x, s’il existe, se trouve dans l’intervalle [M+1..dernier]

La recherche dichotomique consiste à itérer ce processus jusqu’à ce que l’on trouve x ou que l’intervalle de recherche soit vide. 


Procédure RechDicho(T : Tab, x : entier)
Variables
    premier, milieu, dernier : Entier
    trouve : Booléen 
Début
    premier ← 1             // Initialiser l'indice du premier élément
    dernier ← n            // Initialiser l'indice du dernier élément (taille du tableau)
    trouve ← Faux          // Initialiser le statut de recherche à faux

    Répéter
        milieu ← (premier + dernier) div 2   // Calculer l'indice du milieu
        Si (x < T[milieu]) Alors
            dernier ← milieu - 1  // L'élément recherché est dans la moitié gauche
        Sinon
            Si (x > T[milieu]) Alors
                premier ← milieu + 1 // L'élément recherché est dans la moitié droite
            Sinon 
                trouve ← Vrai  // L'élément a été trouvé
            Fin Si
        Fin Si
    Jusqu’à (trouve = Vrai) ou (premier > dernier) // Continuer jusqu'à ce que l'élément soit trouvé ou que la recherche soit épuisée

    Si (trouve = Vrai) Alors
        Ecrire("Indice = ", milieu) // Afficher l'indice de l'élément trouvé
    Sinon 
        Ecrire("Élément introuvable...") // Indiquer que l'élément n'a pas été trouvé
    Fin Si
Fin

Exemple d'Exécution

Imaginons que nous avons un tableau trié :T = [1, 3, 5, 7, 9, 11, 13] avec n = 7. Supposons que nous cherchons l'élément 7.

initialisation :
premier = 1
dernier = 7
trouve = Faux
Première itération :
Calcul de milieu: milieu = ( 1+7)÷2=4
milieu=(1+7)÷2=4
Comparaison :
T[4] = 7, qui est égal à x. Donc, trouve devient Vrai.
Sortie de la boucle :
La boucle se termine, et on affiche : Indice = 4.
