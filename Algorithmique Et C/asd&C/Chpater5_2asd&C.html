Appel d’une fonction

Pour exécuter une fonction, il suffit de faire appel à elle en écrivant son nom suivi des paramètres effectifs (réels) dans le programme principal. Le résultat d'une fonction étant une valeur, devra être affecté à une variable où être utilisé dans une expression, calcul, affichage, test, …etc.

NomVaiable<--Nom_Fonction(pef1, pef2, ...)

Lors de l’appel, il faut respecter le nombre, l’ordre et le type des paramètres.

Exemple1


       En algorithmique
Algorithme CalculTriple
Variables
	nombreEntre , nombreTriple:Entier
    triple(nombre : Entier) :Entier 
    Debut
	    retourner 3 * nombre
    Fin
Debut 
	nombreEntre 0
	nombreTriple 0
	ecrire("Entrez un nombre... ")
	lire(nombreEntre)
	nombreTripletriple(nombreEntre)
	ecrire ("Le triple de ce nombre est", nombreTriple)
Fin


En C
#include 
#include 
int triple(int nombre) // 6
{
return 3 * nombre; // 7
}
void main() // 1
{
int nombreEntre = 0, nombreTriple = 0; // 2
printf("Entrez un nombre... "); // 3
scanf("%d", &nombreEntre); // 4
nombreTriple = triple(nombreEntre); // 5
printf("Le triple de ce nombre est %d\n", nombreTriple); // 8
}//9

Voici ce qui se passe, ligne par ligne:

  1. Le programme commence par la fonction principale main
  2. Il lit les instructions dans la fonction une par une dans l'ordre.
  3. Il lit l'instruction suivante et fait ce qui est demandé (printf).
  4. De même, il lit l'instruction et fait ce qui est demandé (scanf).
  5. Il lit l'instruction… Ah ! On appelle la fonction triple, on doit donc sauter à la ligne de la fonction triple plus haut.
  6. On saute à la fonction triple et on récupère un paramètre (nombre).
  7. On fait des calculs sur le nombre et on termine la fonction. return signifie la fin de la fonction et permet d'indiquer le résultat à renvoyer.
  8. On retourne dans le main à l'instruction suivante.
  9. La fonction main se termine et donc le programme est terminé.

Il faut bien comprendre qu'une fonction reçoit des paramètres en entrée et renvoie une valeur en sortie

Résultat


Entrez un nombre... 10
