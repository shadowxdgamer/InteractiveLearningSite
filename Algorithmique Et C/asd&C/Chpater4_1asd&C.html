Une boucle est une structure qui permet de répéter les mêmes instructions plusieurs fois. Il y a plusieurs façons de réaliser des boucles :

  • TantQue…Faire
  • Répéter … Jusqu’à
  • Pour… Faire

La boucle pour

Les boucles pour sont très utilisées en programmation. Il s'agit de l'instruction:

Syntaxe algorithmique :


Pour compteur De valeur initiale A valeur finale Faire
     <Séquence d’instructions>
FinPour

Principe de fonctionnement

Le compteur (variable de contrôle) prend la valeur initiale au moment d’accès à la boucle puis, à chaque parcours, il passe automatiquement à la valeur suivante dans son domaine jusqu’à atteindre la valeur finale (fig. suivante) :

Remarque

  • Une boucle « pour » peut être exécutée 0, 1 ou n fois
  • On ne peut utiliser la boucle « pour » que si on connaît au préalable combien de fois le traitement sera exécuté (valeur finale – valeur initiale + 1)
  • Dans une boucle « pour », l’évolution du compteur peut se faire dans le sens décroissant (exemple 3)

Exemple1


Pour i de 1 à 10 Faire
    ecrire("Salut mes chers etudiants! ");
FinPour

Cette boucle répète 10 fois l'affichage de «Salut mes chers etudiants ! »

Exemple2


Pour i de 1 à 5 Faire
    Ecrire(i*10)
FinPour

Cette boucle affiche respectivement les nombres 10, 20, 30, 40 et 50.

Exemple3


Pour i de 5 à 1 [pas = -1] Faire
    Ecrire(i*10)
FinPour

Cette boucle affiche respectivement les nombres 50, 40, 30, 20 et 10.

Exemple4


Pour compteur de 1 à 10 Faire
    ecrire("La variable compteur vaut ", compteur ")
FinPour

Dans cet exemple on peut voir que la variable compteur augmente au fur et à mesure dans la boucle, en l'affichant dans le ecrire

Syntaxe en C 


for (compteur; condition; modification du compteur)
{
    liste d'instructions;
}

Remarques

L’intérêt de la boucle for réside dans ce qui se trouve entre les parenthèses. Il y a trois instructions condensées, chacune séparée par un point-virgule :

  • La première est l'initialisation : cette première instruction est utilisée pour initialiser la variable compteur
  • La seconde est la condition : c'est la condition qui dit si la boucle doit être répétée ou non. Tant que la condition est vraie, la boucle for continue.
  • Enfin, il y a l'incrémentation : cette dernière instruction est exécutée à la fin de chaque tour de boucle pour mettre à jour la variable compteur. La quasi-totalité du temps on fera une incrémentation, mais on peut aussi faire une décrémentation (variable--) ou encore n'importe quelle autre opération (variable += 2; pour avancer de 2 en 2 par exemple).

Exemple1


for (i = 0 ; i < 10 ; i++)
{
    printf("Salut mes chers etudiants !\n");
}

Exemple2


for (i = 1 ; i < =5 ; i++)
    printf("%d ",i*10) ;
FinPour

Exemple3


for (i = 5 ; i >=1 ; i--)
	printf("%d ",i*10) ;
FinPour

Exemple4


                  
for (compteur = 1 ; compteur <= 10 ; compteur ++)
 {
    printf("La variable compteur vaut %d\n", compteur);
}
