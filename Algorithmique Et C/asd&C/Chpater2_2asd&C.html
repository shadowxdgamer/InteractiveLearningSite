Les instructions simples

L’instruction d’affectation

Le rôle de cette instruction consiste à placer une valeur dans une variable.

Notation algorithmique
Variable ← Expression

Exemple
L’instruction : A ← 6 signifie « mettre la valeur 6 dans la case mémoire identifiée par A ».
L’instruction : B ← (A + 4) Mod 3 range dans B la valeur 1 (A est toujours égale à 6).

Notation en C
Variable = valeur

Exemple
i = 5 ; // l’affectation de la valeur 5 à i

Remarques

  • La valeur ou le résultat de l’expression à droite du signe d’affectation doit être de même type ou de type compatible avec celui de la variable à gauche.
  • En C, les opérateurs d'affectation composée sont : +=, -=, *=, /=, %=, &=, ^=, |=, <<=, >>=

Pour tout opérateur op, l'expression :
expression1 op= expression2 est équivalente à expression1 = expression1 op expression2. Toutefois, avec l'affectation composée, expression1 n'est évaluée qu'une seule fois.

En C, les opérateurs d'incrémentation ++ et de décrémentation -- s'utilisent aussi bien en suffixe (i++) qu'en préfixe (++i). Dans les deux cas, la variable i sera incrémentée, toutefois dans la notation suffixe, la valeur retournée sera l'ancienne valeur de i, alors que dans la notation préfixe, ce sera la nouvelle.

Exemple

int a = 3, b, c;
b = ++a ; /* a et b valent 4 */
c = b++ ; /* c vaut 4 et b vaut 5 */

L’instruction d’écriture : communication Programme → Utilisateur

Pour qu’un programme présente un intérêt pratique, il devra pouvoir nous communiquer un certain nombre d’informations (résultats) par l’intermédiaire d’un périphérique de sortie comme l’écran ou l’imprimante. C’est le rôle de l’instruction d’écriture. La forme générale de cette instruction est :

Notation algorithmique
Ecrire(expression1, expression2, ...)

Notation en C
printf(expression1, expression2, ...)

Exemple 1
L’instruction : printf("Bonjour\n"); appelle une fonction prédéfinie printf (dans la bibliothèque stdio.h) qui permet d’afficher le message « BONJOUR » à l'écran. \n représente un caractère de fin de ligne.

Remarque : Les caractères spéciaux en C
Les caractères spéciaux sont des lettres spéciales qui permettent d'indiquer qu'on veut aller à la ligne, faire une tabulation, etc. Ils sont faciles à reconnaître : c'est un ensemble de deux caractères. Le premier d'entre eux est toujours un anti-slash \, Cette dernière permet d'avertir le compilateur que le caractère qui suit la \ doit être considéré de façon spéciale.

  • \n : nouvelle ligne
  • \r : retour chariot
  • \t : tabulation horizontale
  • \f : saut de page
  • \v : tabulation verticale
  • \a : signal d'alerte (bip)
  • \b : retour arrière

Exemple 2
L’instruction : printf("a=%d", a); permet d’afficher le contenu d’une variable. On utilise un spécificateur de format, ici %d, pour afficher un entier.

Remarque : Les spécificateurs de format en C

Un «spécificateurs de format » est en fait % suivi d'une lettre (dans l’ exemple précédent , la lettre 'd'). Cette lettre permet d'indiquer ce que l'on doit afficher. 'd' signifie que l'on souhaite afficher un int.

Format Type attendu
%d Affichage en décimal (entiers de type int)
%c Caractère
%ld Affichage en décimal (entiers de type long)
%f Affichage d’un réel (float ou double)
%x Affichage en hexadécimal
%u Affichage en décimal (entiers de type unsigned int)

Exemple
printf("la valeur de a est %d et la valeur de x est %d", a, x);

Autres fonctions de sortie :

  • Affichage d'un caractère : La fonction putchar permet d'afficher un caractère. L'écriture putchar(c) est équivalente à printf("%c \n", c).
  • Affichage d'un texte : La fonction puts permet d'afficher un texte : puts("bonjour") est équivalente à printf("bonjour\n");

L’instruction de lecture : communication Utilisateur → Programme

Dans certains cas, nous devrons transmettre des informations à notre programme via un périphérique d’entrée comme le clavier. Cela sera réalisé par l’instruction de lecture. La forme générale de cette instruction est :

Notation algorithmique
Lire(Variable1, Variable2, ...)
Exemple : Lire(A) signifie lire une valeur et la ranger dans la variable A.

Notation en C
scanf("%f", &x);

La fonction scanf lit des données au clavier. Les variables sont formatées, avec un & devant le nom de la variable pour indiquer son adresse.

Exemple
char alpha;
int i;
float r;
scanf("%c", &alpha); /* saisie d'un caractère */
scanf("%d", &i); /* saisie d'un nombre entier en décimal */
scanf("%f", &r); /* saisie d'un nombre réel */

Remarque
Si l'utilisateur ne respecte pas le format indiqué dans scanf, la saisie est ignorée. Par exemple :

char alpha;
scanf("%d", &alpha);
Si l'utilisateur saisit 97, alpha devient le caractère dont le code ASCII vaut 97. Si l'utilisateur saisit a, sa saisie est ignorée.

Autres fonctions de lecture :

  • La fonction getchar permet la saisie d'un caractère (char) :
    char c;
    printf("ENTRER UN CARACTERE: "); scanf("%c", &c);
  • La fonction getch (bibliothèque conio.h) permet la saisie immédiate d'un caractère alphanumérique :
    printf("ENTRER UN CARACTERE (ATTENTION PAS DE RETURN) ");
    alpha = getch();

La fonction getchar
La fonction getchar est moins gourmande en place mémoire que scanf. Il vaut mieux l'utiliser quand cela est possible.

La fonction getch
La fonction getch, appartenant à la bibliothèque conio.h, permet la saisie clavier d'un caractère alphanumérique. La saisie s'arrête dès que le caractère a été frappé.

On peut utiliser getch de deux façons :

  • Sans retour de variable au programme
    printf("POUR CONTINUER FRAPPER UNE TOUCHE ");
    getch();
  • Avec retour de variable au programme
    char alpha;
    printf("ENTRER UN CARACTERE (ATTENTION PAS DE RETURN) ");
    alpha = getch();
    printf("VOICI CE CARACTERE: %c", alpha);
