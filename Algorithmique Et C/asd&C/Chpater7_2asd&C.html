Le type chaîne de caractères

Déclaration, Affichage et Récupération d'une chaîne

Une chaîne est une suite de caractères. La chaîne ne contenant aucun caractère est appelée chaîne vide.

Déclaration en algorithmique


Variables
c : Caractère
ch : Chaîne
chn : Chaîne[20]

La variable ch peut contenir jusqu’à 255 caractères alors que chn peut contenir au maximum 20. Déclaration en programmation C

Il n'existe pas en langage C de type spécial string. Pour créer un mot ou une phrase, on doit construire une chaîne de caractères. Pour cela, on utilise un tableau de char. 

char NomChaine [Longueur];

Une chaîne de caractères doit impérativement contenir un caractère spécial à la fin de la chaîne, appelé « caractère de fin de chaîne ».Ce caractère s'écrit '\0'.

Exemple 


#include 
int main() {  // Correction de "Void" par "int"
    char chaine[6] = "Salut";  // Tableau de 6 caractères contenant la chaîne "Salut"
    char prenom[100];  // Tableau permettant de stocker le prénom de l'utilisateur
    
    // Affichage de la chaîne grâce au %s du printf
    printf("%s\n", chaine);  // Affiche la chaîne "Salut"
    
    printf("Comment t'appelles-tu ? ");  // Demande à l'utilisateur son prénom
    
    scanf("%s", prenom);  // Lit et stocke la chaîne entrée dans la variable prenom
    
    // Affichage du message avec le prénom de l'utilisateur
    printf("%s, je suis heureux de te rencontrer !\n", prenom);  
    
    return 0;  
}

  3. scanf("%s", prenom); est équivalent à gets(chaine).
