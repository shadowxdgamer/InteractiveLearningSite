Comment utiliser des beans ?

Les beans sont utilisés pour constituer le modèle de l’application (le M de MVC), c’est lui qui va contenir les données, et les traitements à y appliquer. Une fois la classe créée, les beans sont des objets Java sur lesquels on peut faire ces actions :
  • instanciation d'un nouveau bean
  • récupération de la valeur d'une propriété du bean
  • affectation/modification de la valeur d'une propriété du bean
Exemple : Contrôleur 
                                
                                    import model.PersonneBean;
                                    import ....;
                                    @WebServlet...
                                    public class MyServlet extends HttpServlet {
                                        public void doGet(...) {
                                            String nom = request.getParameter("...");
                                            String prenom = request.getParameter("...");
                                            PersonneBean bean = new PersonneBean(); 
                                            bean.setNom(nom); 
                                            bean.setPrenom(prenom); 
                                            request.setAttribute("myBean", bean);
                                            request.getRequestDispatcher("vue.jsp").forward(request, response);
                                        }
                                    }
Utilisation des actions standard : <jsp:useBean>, <jsp:getProperty>, <jsp:setProperty>, et <jsp:forward> 
                                
                                    <html><head>
                                    <title>Ma Vue</title>
                                    </head>
                                    <body>
                                    <jsp:useBean id="myBean" class="PersonneBean" scope="request" />
                                    <jsp:getProperty name="myBean" property="prenom" />
                                    </body>
                                    </html>
Utilisation des Expressions Languages (EL) : 
                                
                                    <body>
                                    <p>${myBean.prenom}</p>
                                    </body>
Ceci rassemble le code Java suivant : 
                                
                                    PersonneBean bean = (PersonneBean) pageContext.findAttribute("myBean");
                                    if (bean != null) {
                                        String prenom = bean.getPrenom();
                                        if (prenom != null) {
                                            out.print(prenom);
                                        }
                                    }
Remarque : Il est possible d’effectuer diverses sortes de tests à l'intérieur d'une expression (&&, ||, ==, !=, <, >, <=, >=, ou la vérification si vide ou null grâce à l'opérateur empty). 
                            
                                <!-- Comparaison d'égalité entre la propriété prenom et la chaîne "Aloui" -->
                                ${myBean.prenom == "Aloui"}
                                <!-- Vérification si la propriété prenom est vide ou nulle -->
                                ${empty myBean.prenom}
