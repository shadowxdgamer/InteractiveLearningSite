Générateur de Quiz
Le Générateur de Quiz est une application web interactive qui vous permet de créer, personnaliser et exporter des quiz.
Vous pouvez créer des questions à choix multiples, à choix unique ou des réponses textuelles avec des extraits de code optionnels.
L'outil prend également en charge la traduction entre l'anglais et le français et propose un design moderne et réactif pour l'édition et l'exportation des quiz.

Fonctionnalités :
Créez des quiz avec plusieurs types de questions :
Réponses textuelles
Questions à choix multiples
Questions à choix unique
Ajoutez des extraits de code optionnels aux questions.
Enregistrez et modifiez des questions existantes.
Exportez les quiz au format HTML interactif.
Interface utilisateur moderne et réactive.
Prise en charge des langues anglaise et française.
Installation et Utilisation :
Prérequis :
Node.js installé sur votre système.
Étapes d'installation :
1.Installez Vite globalement :
npm install -g vite
2.Installez les dépendances:
npm install
3.Lancez le serveur de développement:
npm run dev

Génération pour la production :
Pour générer une version prête pour la production, exécutez :
npm run build

Structure des Fichiers :

components/ : Composants React réutilisables, comme QuestionForm, QuizHeader, etc.
lib/ : Logique pour la génération et l'exportation des quiz.
i18n/ : Fichiers de traduction pour l'anglais et le français.
